# Healthcare Denial Prediction Project

## Overview

This project aims to build a realistic healthcare claims and denial management platform from scratch.

The project will include:

- Relational database design using PostgreSQL
- Synthetic healthcare claims data generation
- SQL analytics
- Denial trend analysis
- Machine learning denial prediction models
- Executive dashboards

## Technologies

- PostgreSQL
- Python
- SQL
- Git & GitHub
- VS Code
- Power BI
